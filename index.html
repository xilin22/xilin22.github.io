Bouncing Ball in PyGame

The object of this game is to have a ball that moves around the pygame screen and have the screen change color every time the ball hits the edges of bthe screen. There is also the strecch challenge of making the ball change color when it hits the sides. To do this, I used pygame methods to create the shapes and move the object. The most challenging part of this project was to calculate the position of the ball, which is needed to set condtion for the ball and screen to change color.
